## 实验简介

### lab1: MIPS程序设计

#### 实验目的

* 熟悉QtSPIM模拟器；
* 熟悉编译器、汇编程序和链接器；
* 熟悉MIPS体系结构的计算，包括
  * MIPS的数据表示；
  * 熟悉MIPS指令格式和寻址方式；
  * 熟悉MIPS汇编语言；
  * 熟悉MIPS的各种机器代码表示，包括
    * 选择结构；
    * 循环结构；
    * **过程调用：调用与返回、栈、调用约定**等；
    * 系统调用。
